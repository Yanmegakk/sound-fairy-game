# Sound Fairy Game - Simple Version

## ゲーム概要

Sound Fairy Gameは、隠れたターゲットを見つけるシンプルなミニゲームです。マウスを動かしてプレイヤーを操作し、ターゲットに接近することでゲームクリアとなります。

## ゲーム仕様

### ゲーム画面
- **解像度**: 800x600 ピクセル
- **背景色**: ダークネイビー（RGB: 20, 20, 30）

### プレイヤー
- **見た目**: 緑色の円（半径 8ピクセル）
- **操作**: マウスカーソルに追従
- **表示色**:
  - 内部色：RGB(100, 255, 100)
  - 枠線色：RGB(150, 255, 150)

### ターゲット
- **見た目**: 画面上に隠れている（見えない）
- **配置**: ゲーム開始時にランダムに配置
  - X座標: 50～750ピクセル
  - Y座標: 50～550ピクセル

### ゲームプレイ

#### 探索フェーズ
- マウスを動かしてプレイヤー（緑色の円）を操作
- ターゲットに接近すると発見判定が進む

#### 発見判定
- プレイヤーとターゲット間の距離が **50ピクセル以内** に近づくと発見
- 発見すると自動的にゲームクリア状態に移行

#### クリア画面
発見時に以下が表示されます：
- 「**Found!**」というテキスト（大きさ: 56pt、色: 黄色 RGB(255, 255, 100)）
- プレイヤー周辺での発光アニメーション（色: RGB(255, 200, 100)）
- 「**Press R to restart**」という再開案内テキスト（大きさ: 16pt、色: グレー RGB(200, 200, 200)）

### 操作方法

| キー/操作 | 機能 |
|----------|------|
| マウス移動 | プレイヤーを移動（画面内に固定） |
| **R キー** | ゲームをリセット（新しいターゲットで再開） |
| **Q キー** | ゲームを終了 |

## セットアップと実行

### 必要な環境
- Python 3.8以上
- UV パッケージマネージャー

### インストール

```bash
# 依存パッケージをインストール
uv sync
```

### 実行方法

```bash
# ゲームを起動
uv run main.py
```

## 使用技術

- **Arcade** (3.3.3): Pythonゲームエンジン
- **NumPy** (2.4.2): 数値計算ライブラリ
- **SciPy** (1.17.1): 科学計算ライブラリ
- **sounddevice** (0.5.5): オーディオ処理用
- **CFFI** (2.0.0): C インターフェース用

## 技術仕様

### 座標系
- 原点: 画面左上
- X軸: 左から右へ増加
- Y軸: 上から下へ増加

### 距離計算
- ユークリッド距離を使用
- 計算式: √((x₂-x₁)² + (y₂-y₁)²)

### グラフィックスレンダリング
- Arcade 3.3.3の描画API を使用
- フレームごとに画面をクリア
- テキストはText オブジェクトでキャッシュしてパフォーマンスを最適化
