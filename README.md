# Sound Fairy Game - 立体音響グリッド探索ゲーム

## 概要
「Sound Fairy Game」は、暗闇（グリッド）の中に隠れた複数のキャラクターの声を聞き分け、指定された「目標話者」の居場所を当てるターンベースの立体音響パズルゲームです。

プレイヤーの現在位置と各話者の座標から**相対距離・角度をリアルタイムに計算し、ステレオパンニングおよび距離減衰（Volume Attenuation）を動的に適用**しています。視覚情報に頼らず、「音の聞こえ方」を頼りに探索を行う点が最大の特徴です。

## 技術的なアピールポイント（音声・DSP処理）

本プロジェクトのコアとなる音声処理は `sound_manager.py` に実装されており、ゲームエンジン既存のオーディオコンポーネントに頼らず、PythonとNumPyを用いた波形レベルのデジタル信号処理（DSP）で空間音響を構築しています。

* **プレイヤーの視線方向を考慮した動的パンニング**
  * プレイヤーの向きベクトルと話者への相対ベクトルから、回転行列（内積）を用いてローカル座標系での相対位置を計算。向いている方向によってリアルタイムに左右の音像定位（パンニング）が変化します。
* **ローパスフィルタ(LPF)による空気吸収シミュレーション**
  * `scipy.signal.butter` を用いて波形にLPFを適用。音源からの距離が遠いほどカットオフ周波数を下げ、「遠くの音は高音域が減衰してこもって聞こえる」という物理現象をシミュレーションしています。
* **プロシージャル足音合成（波形生成）**
  * 事前録音されたSE（WAV）を使用せず、実行時にNumPyで生成したホワイトノイズに対し、指数減衰によるADSRエンベロープとLPFを適用して「ザッ」という足音を数式的に合成・再生しています。
* **ディレイ（反響）処理とリアルタイムミックス**
  * 指定したミリ秒（遅延サンプル数）と減衰率に基づき、元波形に遅延成分を加算することで空間の反響（やまびこ）を表現。
  * 複数のWAV波形データ（最大3音源）に対し、それぞれパンニング・LPFを個別に適用したのち、1つのステレオ配列として加算・正規化（クリッピング防止）して同時再生しています。

## ゲームの遊び方・ルール

### 1. 難易度選択
ゲーム起動時、以下の設定を選択できます。
* **Grid Size (盤面の広さ)**: 3x3 / 5x5 / 10x10
* **Speakers (話者の数)**: 1人 / 2人 / 3人

### 2. ルール
* 盤面上のどこかに話者たちが隠れており、ターゲットがいるマスを音を頼りに推測します。
* ターンごとに足音SEとともに全員が同時に声を発します。
* 「向いている方向」によって声の聞こえる位置が変わるため、立ち止まって周囲を見渡す（SHIFT+矢印キー）ことが攻略の鍵になります。
* ターゲットがいるマスで「Answer (確定)」を行い、正解すればクリアです。

### 3. 操作方法
| キー | アクション |
| :--- | :--- |
| **↑ ↓ ← →** | プレイヤーの移動（移動後に足音SE＋声が再生されます） |
| **SHIFT + 矢印** | 視点移動（移動せず、その場で向きだけを変えて環境音を確認します） |
| **ENTER** | 現在のマスで確定アクションを開始（ダイアログ表示） |
| **Y** | ターゲットがここにいると回答する (Answer) |
| **N** | 回答せずスキップして次のターンへ (Skip) |
| **SPACE** | ターゲットのサンプル音声を再確認 |
| **M** | (リザルト画面) ルールを変更してタイトルに戻る |
| **Q** | ゲームを終了 |

## ディレクトリ構成

```text
sound-fairy-game/
├── main.py              # UI描画・画面遷移・入力制御 (Arcade)
├── game.py              # ゲームロジック・状態管理・座標/視線ベクトル計算
├── sound_manager.py     # 空間音響演算(パンニング/LPF/ディレイ)・プロシージャル合成
├── config.py            # ゲーム定数・DSPパラメータ設定
├── voices/              # 音声アセット (.wav)
└── requirements.txt     # 依存パッケージ

```

## 動作環境・セットアップ

### 必須要件

* Python 3.8 以上

### 依存ライブラリ

* `numpy`: 音声波形の配列操作、相対座標・パンニング・ノイズ生成などの数学的計算
* `scipy`: WAVファイルの読み込み、バターワースフィルタ(LPF)の適用
* `sounddevice`: 計算済みのNumPy配列（ステレオ波形）の非同期再生
* `arcade`: 2Dグラフィック描画、ウィンドウ管理、キー入力制御

### インストールと実行

```bash
# パッケージのインストール
pip install -r requirements.txt
# または uv を使用する場合
# uv pip install -r requirements.txt

# ゲームの起動
python main.py
# または
# uv run main.py

```
