# Sound Fairy Game - 立体音響グリッド探索ゲーム

## 概要
「Sound Fairy Game」は、暗闇（グリッド）の中に隠れた複数のキャラクターの声を聞き分け、指定された「目標話者」の居場所を当てるターンベースの立体音響パズルゲームです。

プレイヤーの現在位置と各話者の座標から**相対距離・角度をリアルタイムに計算し、ステレオパンニングおよび距離減衰（Volume Attenuation）を動的に適用**しています。視覚情報に頼らず、「音の聞こえ方」を頼りに探索を行う点が最大の特徴です。

## 技術的なアピールポイント（音声処理）

本プロジェクトのコアとなる音声処理は `sound_manager.py` に実装されており、以下の音響制御をPythonとNumPyを用いた波形レベルの計算で実現しています。

* **動的ステレオパンニングと距離減衰**
  * プレイヤー座標 $(x_p, y_p)$ と話者座標 $(x_s, y_s)$ の相対ベクトルからユークリッド距離を算出。
  * 距離の遠近に応じたリニアな音量減衰係数（Distance Factor）の適用。
  * X軸の相対位置（$dx$）に基づき、左右チャンネルの音量比率（Left/Right Volume）を計算し、音像定位（ステレオパンニング）を実現。
* **複数音源のリアルタイムミックス（同時再生）**
  * 複数のWAV波形データ（最大3音源）をNumPy配列としてメモリ上にキャッシュ。
  * 各波形に対して個別のパンニング・音量減衰処理を施したのち、すべての波形を加算（ミックス）して1つのステレオ波形を生成。
  * 合成時の音割れ（クリッピング）を防ぐための正規化処理（Normalization）の実装。

## ゲームの遊び方・ルール

### 1. 難易度選択
ゲーム起動時、以下の設定を選択できます。
* **Grid Size (盤面の広さ)**: 3x3 / 5x5 / 10x10
* **Speakers (話者の数)**: 1人 / 2人 / 3人

### 2. ルール
* ゲームが始まると、最初に「目標となる声（ターゲット）」のサンプル音声が再生されます。
* 盤面上のどこかに話者たちが隠れており、ターンごとに全員が同時に声を発します。
* プレイヤーは矢印キーでマスを移動し、ステレオの左右のバランスと音量の大きさから、ターゲットがいるマスを推測します。
* ターゲットがいるマスで「Answer (確定)」を行い、正解すればクリア。不正解の場合はライフ（初期値: 3）が減少します。

### 3. 操作方法
| キー | アクション |
| :--- | :--- |
| **↑ ↓ ← →** | プレイヤーの移動 |
| **ENTER** | 現在のマスで確定アクションを開始（ダイアログ表示） |
| **Y** | ターゲットがここにいると回答する (Answer) |
| **N** | 回答せずスキップして次のターンへ (Skip) |
| **SPACE** | ターゲットのサンプル音声を再確認 |
| **Q** | ゲームを終了 |
| **1 / 2 / 3** | (タイトル画面) グリッドサイズの変更 |
| **W / Q** | (タイトル画面) 話者数の増減 |

## ディレクトリ構成

```text
sound-fairy-game/
├── main.py              # UI描画・画面遷移・入力制御 (Arcade)
├── game.py              # ゲームのロジック・状態管理・座標計算
├── sound_manager.py     # 音声ファイルのロード・立体音響演算・再生制御
├── config.py            # ゲームの定数・パラメータ設定
├── voices/              # 音声アセット (.wav)
└── requirements.txt     # 依存パッケージ

```

## 動作環境・セットアップ

### 必須要件

* Python 3.8 以上

### 依存ライブラリ

* `numpy`: 音声波形の配列操作、相対座標・パンニングの数学的計算
* `sounddevice`: 計算済みのNumPy配列（ステレオ波形）の非同期再生
* `scipy`: WAVファイルの読み込み
* `arcade`: 2Dグラフィック描画、ウィンドウ管理、キー入力制御

### インストールと実行

```bash
# パッケージのインストール
pip install -r requirements.txt
# または uv を使用する場合
# uv pip install -r requirements.txt

# ゲームの起動
python main.py
# または
# uv run main.py
```
