# Sound Fairy Game

音だけを頼りに、隠れたオブジェクトを探すミニゲームの試作です。  
ステレオ音声の定位と距離減衰を用いて、音の方向と距離を表現します。

## 概要
- 画面上のカーソルを操作し、隠れたターゲットの位置を音で探します。
- ターゲットに近づくほど音が大きくなり、左右の位置によって左右音量が変化します。
- 将来的に音声合成（TTS）によるヒント音声の追加を予定しています。

## 使用技術
- Python
- pygame
- numpy
- sounddevice
- scipy

## 実行方法

```bash
uv venv
.venv\Scripts\activate
uv pip install -r requirements.txt
python main.py
````

## 操作方法

* 矢印キー：カーソル移動
* 画面上の緑の点がプレイヤー位置です

## 工夫点

* 距離に応じた音量減衰を実装
* 左右の位置に応じたステレオパンニングを実装
* 音のみでも探索できる設計を意識

## 応用例

* 視覚情報に頼らないゲーム設計
* 位置情報に基づく効果音制御の検証
* サウンドプログラマ向け技術デモ
